Add the official twitter button in your articles

== Description ==

This plugin will help you add the official twitter button in your articles. It's easy to customize and works right after activation.
You will be able to choose where the button shows up inside an article, what type of twitter button to show and what text to submit when a user clicks the button.

== Installation ==

1. Upload the folder `easy-twitter-button` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to your Settings->Twitter Button tab, and update the plugin settings if necessary
